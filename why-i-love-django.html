Why I love Django

Thomas Loiret
  Thu 07 September 2017

Because it prevents you from making bad software design decisions !

Everytime I start having headeach using Django because I try to do something not documented or not discussed on the internet, it often means that my software design is bad.

Django make you think about how you design your code in order to be clean and keep it simple!

Headeach in Django = bad software design ! :)
