Delete all your Google Calendar passed events

Thomas Loiret
  Mon 20 April 2020

Why

I like to put a lot of stuff in my calendar. Lately I've decided to stop using Google products. Calendar is definitively a good one, but you know the saying "if it is free, then you are the product". While I'm still struggling to close my Google account, I've started to clean things up, including my calendar. So I wrote a little script to delete all passed events (and future if you want). I don't need them anymore after all :)

I choose to keep the script quite basic so it can be easily readable. Giving access to your Calendar is not something you should do lightly. Don't hesitate to check how it works before using it ;)

Deletion in 4 steps

1/ Installation

git clone git@github.com:b0uh/google-calendar-events-deletion.git
python -m pip install -r requirements.txt

2/ Get your Google Calendar API credentials

  • Go to https://developers.google.com/calendar/quickstart/python
  • Click on "Enable the Google Calendar API" and choose "Desktop app"
  • Click on "Download client configuration" and save the file credentials.json within the same folder as the python script

3/ Test it

You can run the script without arguments first. By default it will simulate its behaviour and you will see what events should be delete.

You will be asked to authenticate yourself, that's normal.

python events_deletion.py

4/ Just do it (Danger zone!)

python events_deletion.py --confirm-delete

Once the script is done, the events are now in the trash. They will be definitively deleted after 30 days, but if you don't want to wait you can empty the trash at https://calendar.google.com/calendar/r/trash
