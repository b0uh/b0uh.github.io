List of Python web frameworks

Thomas Loiret
  Wed 24 January 2024

Introduction

I'm used to work with Django but it seems good from time to time to take a look at other frameworks to know how they are doing stuff. It can be a good source of aspiration :)

The list

  • Django
    • Django REST Framework - API framework
    • Neopolitan - Quick CRUD views for Django
    • Django Ninja - Django with inspiration from FastAPI
    • django-tastypie - API framework
    • Graphene-Django - Django + GraphQL
  • Flask (micro framework)
    • Quart (async implementation of Flask)
  • FastAPI - API framework based on standard Python type hints
  • AIOHTTP - Asynchronous HTTP Client/Server for asyncio
  • Dash - For ML & data science web apps.
  • CubicWeb - Semantic web application framework
  • Clastic - Functional web framework
  • web2py
  • CherryPy
  • Tornado
  • Masonite
  • Pyramid
  • Bottle
  • Falcon

Not maintained anymore?

  • TurboGears
  • apidoara - API framework using dataclasses and TypedDict annotation for python
  • Hug - API framework
